# 객체 지향 설계와 스프링
스프링은 객체지향 언어가 가진 강력한 특징을 살려내는 프레임 워크이다 즉 스프링은 좋은 객체 지향 애플리케이션을 개발할 수 있게 도와주는 프레임 워크이다.

스프링의 생태계는 위와 같다. 스프링은 어떤 하나가 아니라 여러 스프링 기술을 통틀어 말하기도 한다.
필수적으로 스프링 프레임워크가 있고. 스프링 부트는 최근에는 거의 기본으로 사용되는 추세이다
